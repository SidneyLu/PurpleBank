# PurpleBank -- An Nucleotide Sequence Database 
> This is the final project for course *Database Systems* at NKU-CS

## Tech Stack
- Backend: MySQL
- Host Language: Python
- Frontend: React 

## Database Schematic and Architecture
PurpleBank is designed according to the schematic of NCBI-Gene, with basic
information management and retrieval functions available

## Repository Structure
- database_architecture: establishing the database
- data_preparation: acquiring data sources and insert fasta entries into database
